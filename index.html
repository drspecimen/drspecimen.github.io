DIG InfraCore
Desktop infrastructure monitoring tool for Data centres — real-time UPS, CRAC, and generator oversight with alert management.
SYS // 001
NW.js · Node

A production desktop application built to replace spreadsheet-based infrastructure tracking for Data centres. DIG InfraCore monitors UPS systems, CRAC units, generators, and critical environmental parameters in real time, with an alert engine that flags threshold breaches and logs incident history. Built on NW.js (Chromium + Node.js) for native desktop deployment with no browser dependency. The application significantly reduced the time engineers spend on routine status checks and centralised what had previously been fragmented across paper logs and shared Excel files.

SportSense
AI-assisted sports analysis engine combining live data, manual research, and structured prediction output.
SYS // 002
Python · Flask

An analysis system built to remove guesswork from sports betting research. SportSense pulls fixture, team, and historical data, layers in manually researched context — expected goals, team news, pressing intensity, odds — and passes the combined picture to an AI analysis engine that returns structured predictions: match outcome probabilities, market assessments, and a single ranked "top pick" with a confidence percentage. It now covers football, basketball, and tennis from one Flask interface, with full prediction history and accuracy tracking built in.
